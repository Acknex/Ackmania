 These are the racing adventures of Lotti in dimension JCL-Zeta-Oscar-Romeo-Romeo-Oscar-ACKMANIA.

 How to play:

 ARROW KEYS     -  drive, steer
 SPACE          -  jump / drift
 CTRL           -  weapon / item
 ESC            -  go back
 ENTER / SPACE  -  action
 F5             -  resolution change
 F9             -  mute
 ALT + ENTER    -  fullscreen

 If you don't hear the sounds or some of them, install the OGG drivers (OggDS0995.exe)!
